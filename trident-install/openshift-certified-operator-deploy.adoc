= Trident 커뮤니티 운영자에서 OpenShift 인증 운영자로 전환
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Community Trident Operator에서 Red Hat OpenShift Certified Trident Operator로 전환하려면 Community Operator를 제거한 다음 OperatorHub를 사용하여 인증된 Operator를 설치해야 합니다.

.시작하기 전에
설치를 시작하기 전에link:../trident-get-started/requirements.html["Trident 설치를 위한 환경 준비"] .



== NetApp Trident 커뮤니티 운영자 제거

.단계
. OpenShift 콘솔을 사용하여 OperatorHub로 이동합니다.
+
image::../media/openshift-operator-05.png[설치하다]

. NetApp Trident 커뮤니티 운영자를 찾아보세요.
+
image::../media/openshift-operator-06.png[설치됨]

+

WARNING: *이 연산자에서 모든 피연산자 인스턴스 삭제*를 선택하지 마세요.

. *제거*를 클릭하세요.




== OpenShift 인증 Operator 설치

.단계
. Red Hat OperatorHub로 이동합니다.
. NetApp Trident Operator를 검색하여 선택하세요.
+
image::../media/openshift-operator-05.png[설치하다]

. 화면의 지시에 따라 운영자를 설치하세요.




== 확인

* 콘솔에서 OperatorHub를 확인하여 새로 인증된 운영자가 성공적으로 설치되었는지 확인하세요.
